# 🎮 Tic Tac Toe Game

A modern, feature-rich Tic Tac Toe web application built using **HTML, CSS, and JavaScript**.  
This project includes **Single Player (AI)** and **Multiplayer modes**, along with a premium UI and interactive features.

---

## 🚀 Live Demo
👉(https://bejewelled-tanuki-86e989.netlify.app/)

---

## 📌 Features

- 🎯 **Single Player Mode** (Play against AI)
- 🧠 **Smart AI using Minimax Algorithm** (Unbeatable)
- 🧑‍🤝‍🧑 **Two Player Mode**
- 🏆 **Score Tracking System**
- 🎉 **Winner Popup with New Game Option**
- 🤝 **Draw Detection (No Score Count)**
- 🎨 **Premium Glass UI Design**
- ✨ **Winning Cell Highlight Animation**
- 🎵 **Sound Effects (Click, Win, Draw)**
- 📳 **Vibration Support (Mobile Devices)**

---

## 🛠️ Tech Stack

- **HTML5** – Structure  
- **CSS3** – Styling & Animations  
- **JavaScript (Vanilla)** – Game Logic & AI  

---

## 🧠 AI Logic

The game uses the **Minimax Algorithm**, ensuring that:
- The computer always plays the best move
- The player cannot easily win
- Optimal game decisions are made in every scenario
